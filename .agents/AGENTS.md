# Завершення кожної задачі

Після успішної реалізації задачі завжди виконуй цей порядок:

1. Запусти всі необхідні тести.
2. Запусти перевірки (ruff тощо).
3. Покажи diff.
4. Запропонуй commit message.

Після мого підтвердження:
1. створи Git-коміт;
2. автоматично виконай `git push` до `origin/main`;
3. переконайся, що локальна гілка синхронізована з віддаленою.

Повідом:
- хеш коміту;
- що `origin/main` успішно оновлено.
- Stop.
