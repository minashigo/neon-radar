## Мета етапу

Додати chart до Radar UI для підтвердження вибору монети з ranking.
Ranking залишається PRIMARY view — chart тільки доповнює.

## Що реалізовано

### 1. ChartWidget (`presentation/widgets/chart_widget.py`)

**Pyqtgraph-based chart** з двома зв'язаними plots:

```
┌────────────────────────────────┐
│ Candlesticks + EMA20 + EMA50   │  ← price plot (75%)
│ + Bollinger Bands               │
├────────────────────────────────┤
│ Volume bars                    │  ← volume plot (25%)
└────────────────────────────────┘
```

**CandlestickItem** — custom `QGraphicsObject`:
- Малює wick (high-low line) + body (open-close rectangle)
- Green (#00e676) для bullish (close ≥ open)
- Red (#ff1744) для bearish
- Pre-rendered як `QPicture` для швидкого repaint

**Indicators** — лінії на price plot:
- EMA20 — cyan (#4fc3f7)
- EMA50 — purple (#ab47bc)
- Bollinger — dim grey
- Інші — fallback to text colour

**Volume bars** — `BarGraphItem` на volume plot:
- Color-coded за напрямком свічки
- Slight transparency (alpha=120) для layered look

**Time axis** — `DateAxisItem`:
- Unix timestamps (ms → s)
- Автоматичні labels типу "Jan 15", "Feb 1"

**Performance:**
- Last 200 candles (DEFAULT_VISIBLE)
- NaN values автоматично пропускаються
- Pre-rendered QPicture = швидкий repaint

### 2. DetailPanel — enable "View Chart"

Кнопка "View Chart →" тепер активна після вибору символу.
Викликає `view_chart_requested` signal.

### 3. MainWindow — chart dock

**QDockWidget** з правого боку (Qt-native):
- Можна переміщувати (drag)
- Можна закривати
- Можна змінити область (right/bottom)

**Інтеграція:**
- `view_chart_requested` → show dock + render current selection
- `symbol_selected` → render chart for new symbol
- `klines_ready` → cache latest data → render chart

**Кеш для швидкого рендерингу:**
- `_last_results[symbol]` → AnalysisResult
- `_last_klines[symbol]` → KlineSeries
- `_last_indicators[symbol]` → tuple[IndicatorSeries, ...]

## Архітектурні рішення

### 1. Custom QGraphicsObject для candlesticks

Замість двох QGraphicsItem per candle (wick + body), один
CandlestickItem з pre-rendered QPicture. Один paint call per candle,
а не два. Для 200 candles = 200 paint calls = швидко.

### 2. QDockWidget pattern

Qt-native підхід для "drill-down" вікна. Не модальне — можна тримати
поруч із ranking. Не блокує workflow — можна закрити, коли непотрібно.

### 3. Cache klines + indicators

MainWindow тепер кешує останні дані для кожного символу. При кліку
"View Chart" рендеринг миттєвий (без мережі). При auto-refresh кожні
60с — chart автоматично оновлюється.

### 4. DateAxisItem

Pyqtgraph's `DateAxisItem` дає автоматичні date labels. Конвертуємо
ms → s перед plot.
